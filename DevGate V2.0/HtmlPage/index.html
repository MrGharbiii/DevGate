Les balises html et leurs attributs

Une balise HTML est un élément de base du système de codage HTML utilisé, en grande partie, pour la création d'une page Web. Ce sont les balises HTML qui permettent de hiérarchiser et de structurer le contenu d'une page Web grâce à une mise en forme particulière

Nom attributs

Les balises en pair

Elles s'ouvrent, contiennent du texte, et se ferment plus loin. Voici à quoi elles ressemblent

Nom role
<form>..</form> creation de formulaire
<em>..</em> Texte en italique
<pre>..</pre> Texte préformaté, affichage des espaces et des sauts de ligne
<map>..</map> Carte de définition des zones avec zones
<table>..</table> Définition d'un tableau

Balises les plus courantes

ce sont les balises les plus utilisant dans les pages web

Nom role
<html>..</html> Début et fin de fichier Html
<head>..</head> Zone d'en-tête d'un fichier Html
<title>..</title> Titre affiché par le navigateur (élément de HEAD)
<body>..</body> Début et fin du corps du fichier Html
<div>..</div> creation de division
<img> insertion d image
<p>..</p> creation de paragraphe
<video>..</video> insertion de video
